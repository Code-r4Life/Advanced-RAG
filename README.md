## 📖 About

This repository contains hands-on implementations of modern Retrieval-Augmented Generation (RAG) techniques inspired by research papers and production systems. The focus is on understanding retrieval algorithms, experimenting with different architectures, and implementing them from scratch rather than relying solely on high-level abstractions.
